# v0.3.0 — Phase 2-4: Shader Editing, Diff Engine, Pass Analysis

**Date:** 2026-04-02

## Highlights

- **52 MCP tools** (25 new since v0.2.x)
- Phase 2: Shader hot-editing, mesh/snapshot export, CI assertions (13 tools)
- Phase 3: Dual-capture diff engine with 6 comparison modes (8 tools)
- Phase 4: Pass-level analysis with dependency DAG and unused RT detection (4 tools)

## Phase 2: Shader Editing, Extended Export, CI Assertions (13 tools)

### Shader Hot-Editing
- `shader_encodings` / `shader_build` / `shader_replace` / `shader_restore` / `shader_restore_all`

### Extended Export
- `export_mesh` — OBJ/JSON vertex data
- `export_snapshot` — Complete draw state bundle
- `get_resource_usage` — Resource usage across events

### CI Assertions
- `assert_pixel` / `assert_state` / `assert_image` / `assert_count` / `assert_clean`

## Phase 3: Diff Engine (8 tools)

Dual-capture comparison with LCS-aligned draw matching:
- `diff_open` / `diff_close` / `diff_summary` / `diff_draws`
- `diff_resources` / `diff_stats` / `diff_pipeline` / `diff_framebuffer`

**CLI:** `renderdoc-cli diff FILE_A FILE_B [--draws|--resources|--stats|--pipeline|--framebuffer]`

## Phase 4: Pass-Level Analysis (4 tools)

Hybrid pass enumeration (marker + synthetic gap-fill):
- `get_pass_attachments` — Color/depth attachment query
- `get_pass_statistics` — Per-pass draw/triangle/RT stats
- `get_pass_deps` — Inter-pass resource dependency DAG
- `find_unused_targets` — Dead render target detection with wave ranking

**CLI:** `pass-stats` / `pass-deps` / `unused-targets`

## Notes
- Windows x64 only
- Requires RenderDoc v1.43+ runtime (bundled in release packages)
