First public release of `renderdoc-mcp`, an MCP server for GPU render debugging built on top of RenderDoc.

## Highlights

- 20 MCP tools covering capture opening, event and draw navigation, pipeline inspection, resource inspection, shader analysis, exports, stats, and logs
- Supports `.rdc` captures from D3D11, D3D12, OpenGL, and Vulkan through the RenderDoc replay API
- Ships as a Windows x64 zip with `renderdoc-mcp.exe` and the required RenderDoc runtime files
- Works with Claude Code, Codex, and other MCP clients that use stdio transport

## Included In This Release

- `renderdoc-mcp.exe`
- `renderdoc.dll`
- `d3dcompiler_47.dll`
- `dbghelp.dll`
- `symsrv.dll`
- `symsrv.yes`
- README and license files
- SHA256 checksum file for the release archive

## Getting Started

1. Download and unzip the release archive.
2. Keep all bundled files in the same directory as `renderdoc-mcp.exe`.
3. Configure your MCP client to launch `renderdoc-mcp.exe`.
4. Open a RenderDoc capture with `open_capture`, then inspect draws, resources, shaders, and render targets with the included tools.

## Notes

- Windows x64 only for now
- This release packages RenderDoc v1.36 runtime files
